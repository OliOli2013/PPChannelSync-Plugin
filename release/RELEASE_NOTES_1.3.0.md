# PP Channel Sync 1.3.0 — Release notes

Wydanie naprawia powtarzający się błąd odczytu paczki kontrolnej bez pliku o dokładnej nazwie `lamedb`. Od tej wersji poprawnie obsługiwane są `lamedb` i `lamedb5`, również w zagnieżdżonych archiwach.

Dodatkowo wprowadzono sprawdzaną kopię bezpieczeństwa, automatyczne cofnięcie nieudanej korekty, bezpieczne rozpakowywanie, bardziej szczegółową diagnostykę i kontrolę SHA256 aktualizacji.

## Plik do wydania

`enigma2-plugin-extensions-ppchannelsync_1.3.0_all.ipk`

## Zalecenie

Przed publikacją dla wszystkich użytkowników wykonać test na co najmniej jednym obrazie Python 2 oraz jednym obrazie Python 3.
