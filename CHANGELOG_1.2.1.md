# PP Channel Sync 1.2.1

Zmiany względem 1.2.0:

- dodano trzecie źródło kontroli: **GioppyGio / OpenVisionE2**,
- dodano obsługę repozytorium `OpenVisionE2/GioppyGio-settings`,
- dodano wybór pakietów GioppyGio:
  - Mono 13E,
  - Dual 13E + 16E,
  - Dual 13E + 19E,
  - Dual 5W + 13E,
  - Dual 9E + 13E,
  - Trial,
  - Quadri,
  - Motor 75E - 45W,
- pozostawiono dotychczasowe źródła bez zmian:
  - Standard / Vhannibal,
  - Alternatywne / Ciefp,
- GioppyGio działa jako dodatkowa baza kontrolna, a nie jako instalator gotowej listy,
- zachowano obecną logikę PP Channel Sync: raport, bezpieczna korekta, backup, ochrona IPTV/DVB-T/DVB-C.
