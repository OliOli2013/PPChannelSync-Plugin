# Migracja do PP Channel Sync 2.1

1. Zainstaluj wersję 2.1.1 i uruchom ponownie GUI Enigma2.
2. Wybierz wszystkie używane pozycje orbitalne.
3. Najpierw wykonaj `Raport bez zapisu`.
4. Sprawdź liczbę nowych kanałów przewidzianych do dopisania.
5. Uruchom `Bezpieczną korektę`.

Wersja 2.1.1 zachowuje wszystkie funkcje 2.1.0 i rozpoznaje stare bloki `nowe kanały - PP Channel Sync`, zachowuje poprawne kanały i przebudowuje je do jednego kontrolowanego bloku z markerem końcowym. Wtyczka nie usuwa automatycznie zwykłych kanałów użytkownika.
